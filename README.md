# Dhruv Sharma — Portfolio

A personal portfolio built with React + Vite.

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 📦 Build & Deploy

```bash
npm run build     # Build for production
npm run deploy    # Deploy to GitHub Pages
```

## 🔧 Setup

1. Copy `.env.example` to `.env`
2. Fill in your EmailJS credentials (see emailjs.com)
3. Replace `public/resume.pdf` with your actual resume
4. Replace `src/assets/images/photo.jpg` with your photo
5. Update `src/data/` files with your real info
